# UnlockBox PWA

PWA para alquiler de cajas decodificadoras. HTML/CSS/JS puro.

## Uso
1. Sube los archivos a un repositorio de GitHub.
2. Activa GitHub Pages (Settings → Pages → main branch).
3. Abre la URL — funciona como app instalable.

## Características
- Catálogo de cajas con imagen, precio, descripción y links sociales (WhatsApp, Facebook, Telegram).
- Modal de información detallada con links de descarga.
- Registro de clientes con estado (solicitado/alquilado/devuelto) y exportación CSV.
- Panel admin para crear/editar/eliminar cajas (con subida de imagen local).
- PWA instalable, funciona offline, responsive.
- Persistencia en localStorage del navegador.

Colores: negro + rojo-marrón. Fuente: Open Sans / Segoe UI (estilo Facebook).
